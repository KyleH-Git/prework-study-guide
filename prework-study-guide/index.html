Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements are the different sections of a website like paragraphs, lists, headers, imgs etc.
  • An element is both opening and closing tags with body included.
  • Tags are just the opening or closing sections of an element.
  • Html has a base style and functionality, CSS is a better way to format and style web applications.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline styling is style defined in the element tag and has to be changed individually.
  • Internal stylesheets are located in the header element of an html file and styles all elements of that type in the file.
  • External stylesheets are .css files connected to an html file where elements can be styled from.
  • Class selectors are used to style specific elements in an html file from an external style sheet.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git is useful for collabrating on a project with others without overwriting each others work.
  • Properly managing git workflow will leave a clear history of changes made and make tracking bugs easier.
  • Making sure to update the main branch to your local directory before making changes ensures it's up to date before making any changes.
